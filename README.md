# 📊 Stock Market Dashboard (Streamlit)

An interactive financial dashboard built with Python and Streamlit to analyze stock market data in real time.

## 🚀 Overview

This project allows users to explore stock price trends and trading volume for major companies using an intuitive and interactive web interface.

The dashboard fetches real-time data using the Yahoo Finance API and provides key insights through dynamic visualizations.

## ✨ Features

* 📈 Interactive stock price visualization
* 📊 Trading volume analysis
* 📅 Custom date range selection
* 📉 Moving Average (MA50) indicator
* 📌 Key metrics (latest, highest, lowest price)
* ⚡ Fast and responsive UI with Streamlit

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Plotly
* yFinance


## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/stock-market-dashboard-streamlit.git
cd stock-market-dashboard-streamlit
```

Create a virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Run the App

```bash
streamlit run app.py
```

Then open your browser at:

```
http://localhost:8501
```
